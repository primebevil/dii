- [Capability-Routing](./Capability-Routing/): the router and scheduler core
- [Discovery](./Discovery/): federated node and pod discovery
- [Trust](./Trust/): identity, reputation, policy

## Guiding constraints

Local-first always, graceful degradation never to zero, no single point of control, open weights only, and sovereignty as a first-class routing input.
